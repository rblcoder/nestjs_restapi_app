## Description

Learning [Nest](https://github.com/nestjs/nest) with https://www.prisma.io/blog/nestjs-prisma-rest-api-7D056s1BmOL0.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
